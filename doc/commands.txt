This file lists all commands used to create models/views/controllers:

Sponsor: 
    rails g active_scaffold Sponsor name:string
Institution: 
    rails g active_scaffold Institution name:string
Level: 
    rails g active_scaffold Level name:string
Course: 
    rails g active_scaffold Course name:string level:references institution:references
Student: 
    rails g active_scaffold Student name:string cpf:string
Course-Student:
    rails g migration CreateCoursesStudents
    (followed by the edition of the created migration. Check the migration for future join tables)
